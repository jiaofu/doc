## 不可变集合


对应于元组（tuple）与列表（list）的关系，对于集合（set），Python提供了一种叫做不可变集合（frozen set）的数据结构。

使用 frozenset 来进行创建
```
s = frozenset([1, 2, 3, 'a', 1])
```
